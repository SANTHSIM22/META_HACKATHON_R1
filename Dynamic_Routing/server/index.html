Dynamic Route Optimizer

A dynamic logistics simulation for LLMs and RL agents to manage delivery fleets, prioritize package deadlines, and instantly reroute trucks around unpredictable obstacles.

Getting Started

Use the Python client for programmatic access:

from client import DynamicRouteEnv
from models import DynamicRouteAction

env = DynamicRouteEnv.from_docker_image("dynamic-route-env:latest")
result = env.reset()

action = DynamicRouteAction(route_updates=[{
    "truck_id": "TRK_100",
    "new_route_order": ["Node_50_50", "Node_20_20"]
}])
result = env.step(action)
env.close()

How It Works

  • Procedural Maps: Each reset() generates a fresh node graph connected via Manhattan distance.
  • Strict Validation: Proposed nodes must exactly match those generated in the current episode.
  • Unexpected Events: Road segments can become blocked; trucks stall and require rerouting.
  • Scoring: Score (0.0–1.0) rewards on-time deliveries and penalizes inefficient routes.

Using the Web UI

1
Initialize a session. Click "Initialize Session" — opens Raw Response tab with the reset payload.
2
Copy valid IDs. From the Fleet table, note a valid truck_id. Check Raw Response for valid node names under distances.
3
Send a step. Paste your JSON into the Dispatch Action textarea and click Send Step.

Common Errors

  • route_updates is empty  Sent {} with no updates array.
  • Truck 'TRK_999' not found  Truck ID doesn't exist; copy it from the Fleet table.
  • Node 'Node_99_99' not found in city map  Node wasn't generated this round.
  • TRK_143 blocked: Node_A to Node_B. Reroute!  Route crosses a storm blockage.

Schema Reference

Action

  • route_updates — List of objects: truck_id + new_route_order (array of node strings).

Observation

  • time_step — Current simulation tick.
  • trucks — Array of truck states: location, assigned packages, pending route.
  • packages — Status and deadlines per package.
  • event — traffic_delays and blocked_edges for real-time avoidance.
  • distances — Full procedurally generated distance matrix.
